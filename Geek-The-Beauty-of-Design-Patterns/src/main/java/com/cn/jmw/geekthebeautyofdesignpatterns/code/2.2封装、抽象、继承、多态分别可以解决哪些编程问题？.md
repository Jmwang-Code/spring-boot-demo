# 1. 封装（Encapsulation）
作用定义：信息隐藏或者数据访问保护。
意义：防止用错，更简单的使用和调用函数。

# 2. 抽象（Abstraction）
定义：那抽象就是讲如何隐藏方法的具体实现，让使用者只需要关心方法提供了哪些功能，不需要知道这些功能是如何实现的。
意义：只关注功能点不关注实现的设计思路

# 3.继承（Inheritance）
定义：继承是用来表示类之间的 is-a 关系
意义：继承主要是用来解决代码复用的问题。

# 4.多态（Polymorphism）
定义：可以理解为在不同条件下，所实现同一个类型的不同表现，同样是飞行，鸟和飞机是不一样的。多态特性的实现方式，除了利用“继承加方法重写”，还有接口类
意义：提高扩展性、提高复用性
