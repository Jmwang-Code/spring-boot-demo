1. 常数时间的操作 ： + - * / %  >> >>> << | & ^ 数组寻址 o(1)
2. 对数时间的操作 ： 二分查找 o(logn)
3. 线性时间的操作 ： 顺序查找 o(n)
4. 线性对数时间的操作 ： 归并排序 o(nlogn)
5. 平方时间的操作 ： 选择排序 o(n^2)
6. 立方时间的操作 ： 二维矩阵乘法 o(n^3)
7. 指数时间的操作 ： 汉诺塔 o(2^n)
8. 阶乘时间的操作 ： 旅行商问题 o(n!)
   时间复杂度计算 ：f(n) = an^2 + bn + c =>  O(n^2)

稳定性：排序后两个相等的数相对位置不变
可以做到排序稳定的：冒泡排序、插入排序、归并排序和基数排序
